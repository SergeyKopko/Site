перейти в директорию -  cd educational_institution (там где храится файл manage.py)


после командой python manage.py runserver запустить сервер и перейти по ссылке нашего сайта

если хотим зайти в admin панель пишет  в поисковой строке браузера - http://127.0.0.1:8000/admin/ 

вот логин и пароль от админ панели: admin admin

если хотите создать свою , то пишете в терминале в PyCharm команду - python manage.py createsuperuser

Когда войдете в админ панель:

 Можете выгружать информацию в zip файл о каждом студенте!

Можете добавлять карточку блоги / статьи / новости на сайт;

Можете добавлять карточку об учителях на сайт;

Можете добавлять карточку о достижениях и лицензях учреждения на сайт ;

Можете добавлять карточку о Платных услугах учреждения на сайт ;
